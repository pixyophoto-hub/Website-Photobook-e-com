Telegram — Import Order
Post order dalam Telegram guna arahan /order (lampirkan resit sekali) → terus masuk sistem sebagai sumber Telegram. Editor boleh proses macam order biasa.
Taip /id (dalam chat, grup, atau channel) untuk dapat ID. Untuk channel: jadikan bot admin channel & tambah ID channel di sini. Biar kosong = benarkan sesiapa (tak disyorkan).
Webhook: belum disambung
Format order dalam Telegram:
/order
Nama: Siti Aminah
Telefon: 0123456789
Item: Photobook 8x8 Hardcover | 1 | 125
Item: Add-on Layout | 1 | 4
Alamat: No 12, Kajang, Selangor
Nota: dah bayar penuh
Item = nama | kuantiti | harga. Lampirkan gambar resit sekali — akan disimpan.
